Standard Providers
This is how a standard provider looks like. 
		
			@Module({
				controllers: [ProductController],
				providers: [ProductService]
			})
The syntax providers: [ProductService] is short-hand for the more complete syntax 
		
			providers: [{
				provide: ProductService, //Token
				useClass: ProductService //Class
			}];
The token ProductService is associated with the class ProductService.
Other Providers
Value Providers
		
			const ProductServiceMock = {
				//mock implementation
			};
			@Module({
				imports: [ProductModule],
				providers: [{
					provide: ProductService,
					useValue: ProductServiceMock
				}]
			})
Non-class Based Value Providers
		
			@Module({
				providers: [{
					provide: 'HOST_ADDRESS',
					useValue: HostAddress
				}]
			})
		
	  
		
			@Injectable()
			export class ProductRepository{
				constructor(@Inject('HOST_ADDRESS') hostAddress: HostAddress){}
			}
