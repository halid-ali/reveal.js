What is AsciiDoc?
  • Antora's native content markup language
  • open source
  • readable, extensible and easy to learn
  • supports all the structural and semantic elements for writing documentation
  • Asciidoctor.js is the processor that converts AsciiDoc content to HTML
for more information: https://docs.antora.org/antora/latest/asciidoc/asciidoc/

AsciiDoc is a lightweight and semantic markup language primarily designed for technical documentation.

It is open-source.

Ideal for writing documentation because it is easy-to-learn and readable and extensible

Supports structural and semantic elements to produce a variety of rich output formats

Its parser Asciidoctor.js, loads, parses and converts documents extremely quickly
