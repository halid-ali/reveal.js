Antora supports search functionality that allows users to easily find the content they need within the documentation. The search functionality is based on Lunr.js, a JavaScript search engine that provides fast and accurate search results.

To enable search in Antora, you need to include the @antora/search component in your Antora playbook file. Once the component is included, Antora will automatically create a search index and provide a search bar.

Antora provides support for multilingual documentation, allowing you to create documentation in multiple languages and provide a language switcher to enable users to switch between languages.

To enable multilingual support in Antora, you need to include the @antora/i18n component in your Antora playbook file.

Antora provides support for user authentication, allowing you to restrict access to your documentation to specific users or groups.

To enable user authentication in Antora, you need to include the @antora/auth component in your Antora playbook file. This component allows you to define authentication providers, such as LDAP or GitHub. Once authentication is set up, Antora will require users to log in before they can access your documentation.
