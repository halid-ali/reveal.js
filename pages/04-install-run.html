Install and Run
Install Node.js
        $ nvm install 18.10.0
Install Antora
        $ node -e "fs.writeFileSync('package.json', '{}')"
$ npm i -D -E @antora/cli@3.1 @antora/site-generator@3.1
Verify Antora
        $ npx antora -v
@antora/cli: 3.1.2
@antora/site-generator: 3.1.2
Run Antora
        $ npx antora --fetch antora-playbook.yml

At very first, node js must be installed. To install it, run this command. You can use the latest version of the node.js

After that you need to install Antora. The first line will create a package.json in your antora project root folder. And the second one will install the Antora CLI and site generator.

When the installation complete, you can verify the installation by checking the version of the Antora. And you should have this output.

After all, by using the following command, you can run Antora on your local.
