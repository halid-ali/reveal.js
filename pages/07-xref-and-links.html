Xref and Links
An AsciiDoc xref macro is used to create a source-to-source cross reference from a resource to a publishable resource. 
//Page links within a module
xref:mypage.adoc[My Page]
//Page links between modules
xref:modulename:mypage.adoc[My Page]
//Page links between versions
xref:version@component:modulename:mypage.adoc[My Page]

Xrefs are used to create links to the pages within module, between modules and even between versions.
