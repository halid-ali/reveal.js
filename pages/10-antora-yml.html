antora.yml
antora.yml is a configuration file that serves two crucial functions.
  1. It signals to Antora that it should find a sibling modules directory that contains source files that belong to a component version.
  2. It provides the required and optional component version metadata that Antora assigns to the source files 
        name: antora-demo
version: v4.0
