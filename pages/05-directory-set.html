Antora collects and processes content source files according to a set of hierarchical files and directories it expects to find at a content source root.

  1. Content source root.
  2. The descriptor file which is required at each content root
  3. It must be located at the same level as antora.yml file
  4. ROOT is reserved directory name and must be in all uppercase
  5. Pages family directory (reserved directory name)
  6. Partials family directory (reserved directory name)
  7. Navigation file
  8. It won’t be processed because located outside the modules
