Getting Started with AsciiDoc

Install AsciiDoc

sudo apt-get install asciidoc
brew install asciidoc

Setup your environment

my_doc.adoc

Write your first document

= My First AsciiDoc Document
== Introduction

Convert your AsciiDoc to HTML

asciidoctor my_doc.adoc

On Linux, you can use the package manager, on macOS, you can use Homebrew to install AsciiDoc.
On Windows, you can download the AsciiDoc package from the AsciiDoc website and install it manually.

Create a new directory for your documentation project. Create a new file with a .adoc extension. Open the file in a text editor.

click for next

Start your document with a heading by using = followed by a space and the heading text. Add a paragraph by simply typing the text. Add a second-level heading by using == followed by a space and the heading text. And add another paragraph.

click for next

Run asciidoctor in the terminal. This will create a new file called my_doc.html in the same directory. Congratulations! You've just created your first AsciiDoc document.
