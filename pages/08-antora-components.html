Antora Components
  • Modules
    They are self-contained units of documentation that contains pages and assets. They can be versioned and reused across multiple documentation projects.
  • Pages
    They are the individual documentation files that make up a module. They can be written in AsciiDoc or another markup language supported by Antora.
  • Navigation
    Antora automatically generates a navigation menu based on the page hierarchy defined in your documentation.
  • UI Components
    Antora provides a set of UI components (such as buttons, tabs, and accordions).

All of these components work together to create a unified documentation experience for your users. By breaking your documentation down into modular pieces and organizing it in a hierarchical structure, you can make it easier for users to find the information they need.
