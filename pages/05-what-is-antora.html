What is Antora?
  • An open-source documentation platform that uses AsciiDoc
  • Designed to help organizations manage and publish documentation
  • Modular documentation: allows you to break down your documentation into modules
  • Site generation: can generate static websites from your documentation modules
  • Navigation and search: includes features for creating navigation menus and search functionality
  • Customization: support for custom themes, templates, and styles
  • Integration: can be integrated with other tools and systems

Antora is an open-source documentation platform that uses AsciiDoc as its markup language.

It is designed to help organizations manage and publish documentation across multiple sites and versions, making it easier for teams to collaborate and keep documentation up-to-date.

Modular documentation: Antora allows you to break down your documentation into modules, which can be versioned and managed separately. This makes it easier to maintain and update documentation over time.

Site generation: Antora can generate static websites from your documentation modules, making it easy to publish documentation to the web. It supports a variety of site generators, including AsciiDoctor Site, Assemble, and Hugo.
