What is Antora?
  • eases to create and publish documentation for the tech writers
  • transforms organized AsciiDoc contents in git repos into a website
  • provides separation of content, configuration and presentation
  • automates the compilation, validation and publishing processes
  • reuses the shared materials to conform DRY

Briefly, Antora is a site generator for tech writers that provides documentation pipeline for engineering teams.

It publishes documentation sites composed in AsciiDoc and sourced from multiple versioned repositories.

It separates the concerns of configuration, presentation and the content.

With help of the Antora CLI, it helps you to automate the processes.

By using the shared materials, it conforms to DRY principle.

These are the some practices that Antora includes. For more you can look into the its documentation website. Let's go with AsciiDoc.
